# TFG - Plataforma de estadísticas deportivas con IA

## Descripción
Aplicación web para consulta de estadísticas deportivas con soporte de consultas en lenguaje natural mediante IA.

## Tecnologías
- Backend: Python (FastAPI)
- Frontend: React
- Base de datos: PostgreSQL
- ETL: SoccerData

## Arquitectura (inicial)
- Backend API
- Base de datos relacional
- Servicio ETL
